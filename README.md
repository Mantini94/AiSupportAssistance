# AiSupportAssistant

An AI-powered customer support platform that automates ticket processing, email communication and response generation using React, Supabase, n8n and OpenAI.

---

## Overview

AiSupportAssistant is a modern helpdesk application designed to reduce manual customer support work by combining workflow automation with artificial intelligence.

The system automatically receives customer emails, analyzes their content with AI, classifies each ticket, evaluates urgency and business risk, generates professional reply suggestions and allows support agents to review, edit and send responses from a single dashboard.

The project demonstrates a complete AI workflow, from incoming customer message to automated analysis and assisted response generation.

## Installation

Clone the repository

```bash
git clone https://github.com/Mantini94/AiSupportAssistant.git
```

Install dependencies

```bash
npm install
```

Run development server

```bash
npm run dev
```

---

## Environment Variables

Create a `.env` file and configure:

```env
VITE_SUPABASE_URL=
VITE_SUPABASE_ANON_KEY=
```

Configure your n8n webhooks inside:

```
src/lib/ticketHelpers.js
```
